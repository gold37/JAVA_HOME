OracleXE -> setup.exe 설치 후
cmd 창 켜서

C:\Users\user1>sqlplus /nolog

SQL> connect system/eclass(비밀번호)
▶ conn만 써도 됨
SQL> conn system
▶ 암호 안보이게 칠 수 있음

SQL> show user
▶ 현재 오라클 서버에 연결된 사용자명 알려줌

SQL> disconnect
▶ 연결 끊음, 여기도 disconn만 써도 됨

SQL> host cls
▶ 화면 깨끗하게 지우기

SQL> shutdown immediate
▶ 오라클 서버를 중지(끊음)

SQL> conn sys/eclass as SYSDBA
▶ 관리자 계정인 sys로 연결을 하려면 반드시 'as SYSDBA'를 붙여야 한다.
SQL> conn sys as sysdba
▶ 이렇게 써도 됨

※ SYS랑 SYSTEM 차이는 반장, 부반장 느낌's

SQL> startup
▶ 중지됐던 오라클 서버를 다시 켬 ---> SYS만 할 수 있음 (SYSTEM은 아예 불가능)
